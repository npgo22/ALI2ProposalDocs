---
title: LR2021
description: Long-range LoRa / (G)FSK / FLRC transceiver for the telemetry link.
---

The LR2021 is the long-range radio transceiver used for the telemetry link between the ground station and the vehicle, and on the chickenstick locator. It supports LoRa, (G)FSK, and FLRC modulations.

:::caution
The detailed selection rationale for this part is still being written up.
:::

---
title: LAN8742A
description: Ethernet PHY used on the ground-station boards.
---

The main reason for going with this one is because it was originally used on the STM32H563 Nucleo board, from which we were originally trying to base the thermocouple-v1 board on. This later proved to be a terrible idea for various reasons, however, the chip still stuck as it is completely suitable for our purposes.
